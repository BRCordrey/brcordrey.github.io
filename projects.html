Chess Robot

This is a program that I wrote in Python to display a game of chess being played with a robotic arm that can move the pieces. The intent with this project was to build a physical chess board with a 2 joint arm under the table that would allow the board to play itself. The program uses "FEN" Strings to decode some of the most famous games ever played, I had to do some tweaking to the "Fen" strings to make them more readable to the user and also the program. This project gave me a good introduction for programing to the physical world as well as some graphics and physics experience with the robot arm. Some day I hope to build the board once I have more experience with electronics.
